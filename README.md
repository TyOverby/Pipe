Pipe
====

[Download Build](https://github.com/TyOverby/Pipe/releases)


## How to use

### Linux

    cd vulcan_0.4/pack/bin/
    ./vulcan < inputFile.vlcn > outputFile.latex
    
### Windows
Using Windows PowerShell

    cd vulcan_0.4\pack\bin
    Get-Content .\inputFile.vlcn | .\vulcan.bat > outputFile.latex
