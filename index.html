Vulcan

A Document Preparation Language

Vulcan is a markup language focusing on simplicity that compiles down to LaTeX. The main goal of Vulcan is to remove as much boilerplate and non-essential markup as possible, leaving you with source files that look remarkably like their generated output.

Hello Vulcan

Getting some simple text on screen is a common exercise when it comes to learning a new programming language. Let's do that now. Open up your text editor of choice and type in the following bit of text: 


Hello World

Running this example through Vulcan and then through pdflatex will yield a document only containing the contents "Hello World".
